# Hand Gesture Mouse

Control your computer mouse using just your hand and webcam!

## How it works
- Uses your webcam to detect hand movements
- MediaPipe detects hand landmarks
- Moves the mouse cursor based on your index finger position

## Requirements
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## How to run
1. Install requirements: pip install opencv-python mediapipe pyautogui
2. Run in Terminal: py -3.11 "$env:USERPROFILE\Desktop\run.py" 
3. Show your hand to the webcam!
